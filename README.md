# РосТендер → deal.csv

Конвертер выгрузок из агрегатора тендеров **РосТендер** (Excel) в единый CSV-файл для импорта сделок в CRM.  
Приложение работает полностью в браузере — все данные остаются на вашем устройстве, ничего не отправляется на сервер.

## Возможности

- Загрузка **нескольких файлов** `.xlsx` одновременно (drag & drop или через диалог выбора).
- Для каждого файла отображается статус обработки:
  - ⏳ **Ожидание / Обработка** — анимированный спиннер.
  - ✅ **Готово** — зелёная галочка, показано количество найденных тендеров.
  - ❌ **Ошибка** — красный крестик и сообщение об ошибке.
- **Прогресс обработки** внизу списка: сколько файлов обработано из скольких.
- **Дедупликация** тендеров по столбцу **«Номер тендера»**:
  - Если в разных файлах встречается одинаковый номер, в итоговый CSV попадает только первый (в порядке загрузки).
  - Строки без номера считаются уникальными и не удаляются.
- **Автоматическое извлечение гиперссылки** из столбца «Ссылка на источник» (если она есть как ссылка Excel).
- **Форматирование даты** окончания в `ДД.ММ.ГГГГ чч:мм:сс`.
- **Форматирование суммы** в `100000.00` (два знака после запятой, разделитель точка).
- Возможность **удалить любой файл** из списка до скачивания.
- Подсчёт **общего количества строк** и **уникальных тендеров** после дедупликации.

## Как использовать

1. **Откройте страницу** с приложением (или запустите `index.html` локально).
2. **Загрузите файлы**:
   - Перетащите один или несколько Excel-файлов в область «Перетащите файлы или нажмите для выбора».
   - Или нажмите на эту область и выберите файлы стандартным диалогом.
3. **Дождитесь окончания обработки**:
   - Для каждого файла появится спиннер, который сменится на галочку (или крестик при ошибке).
   - Внизу будет показано, сколько файлов обработано.
4. **Проверьте статистику**:
   - Общее количество строк (тендеров) во всех файлах.
   - Количество уникальных тендеров после удаления дубликатов.
5. **При необходимости удалите файлы** — нажмите крестик справа от имени файла.
6. **Нажмите «Скачать CSV»** — будет сгенерирован файл `deal.csv` со всеми уникальными тендерами.
7. **Импортируйте** полученный CSV в вашу CRM.

## Требования к входным файлам

- Формат: **`.xlsx`** или **`.xls`** (первый лист).
- Заголовки столбцов должны находиться **в 3‑й строке** (индекс 2) и содержать следующие названия **точно** (регистр и пробелы важны):
  - `Предмет тендера`
  - `Заказчик`
  - `Контактное лицо организатора` (необязательно)
  - `Начальная цена, руб`
  - `Ссылка на источник`
  - `Дата окончания (МСК)`
  - `Номер тендера` (необязателен, но без него не работает дедупликация)
- Данные тендеров начинаются с **4‑й строки**.
- В файле могут присутствовать другие столбцы — они игнорируются.

## Выходной файл `deal.csv`

- Разделитель: **`;`**
- Кодировка: **UTF-8 с BOM** (для корректного отображения кириллицы в Excel).
- Структура заголовка соответствует полному шаблону импорта CRM (~240 колонок).

### Соответствие полей

| Поле `deal.csv`                        | Источник в РосТендере            | Примечание                         |
|----------------------------------------|----------------------------------|------------------------------------|
| Название сделки                        | Предмет тендера                  |                                    |
| Компания                               | Заказчик                         |                                    |
| Контакт                                | Контактное лицо организатора     |                                    |
| Валюта                                 | –                                | всегда `RUB`                       |
| Воронка                                | –                                | всегда `Тендеры`                   |
| Стадия сделки                          | –                                | всегда `Распределение`             |
| НМЦК                                   | Начальная цена, руб              | формат `123456.78`                 |

| окончание срока подачи заявок          | Дата окончания (МСК)             | формат `ДД.ММ.ГГГГ чч:мм:сс`      |
| Все остальные поля                     | –                                | пустые                             |

## Примечания и ограничения

- Если в файле отсутствуют **обязательные столбцы** (кроме «Номера тендера» и «Контактного лица»), обработка этого файла завершится ошибкой.
- Дата из столбца «Дата окончания (МСК)» должна быть распознана Excel как дата, иначе она останется в исходном текстовом виде.
- Гиперссылка извлекается **только из столбца «Ссылка на источник»**, другие ячейки с ссылками игнорируются.
- При дедупликации сравниваются **строковые значения** номера тендера после обрезки пробелов. Пустые номера считаются разными и не объединяются.
- Прогресс обработки файлов и статистика тендеров обновляются **автоматически** по мере завершения каждого файла.
- Кнопка «Скачать CSV» становится активной, как только появляется хотя бы один уникальный тендер.
