РосТендер → deal.csv
Конвертер выгрузки из агрегатора тендеров РосТендер (Excel‑файл) в формат CSV для импорта сделок в CRM или воронку продаж.
Приложение работает полностью в браузере, никакие данные не отправляются на сервер.

Возможности
Загрузка файла .xlsx через Drag-and-Drop или выбор файла.

Автоматическое извлечение данных из первого листа:

заголовки определяются по 3‑й строке (индекс 2);

данные тендеров берутся с 4‑й строки и до конца.

Сопоставление столбцов по точному названию (пробелы и регистр учитываются):

Предмет тендера → Название сделки

Заказчик → Компания

Контактное лицо организатора → Контакт

Начальная цена, руб → Сумма (преобразуется в 100000.00)

Ссылка на источник → ссылка (извлекается гиперссылка, если есть)

Дата окончания (МСК) → Срок подачи (формат ДД.ММ.ГГГГ чч:мм:сс)

Стадия сделки всегда «Обсчёт», валюта RUB.

После обработки выводится количество найденных тендеров.

Генерация CSV с разделителем ;, корректное экранирование полей (кавычки, переносы строк).

Скачивание готового файла с именем deal.csv.

Как использовать
Откройте страницу приложения (или запустите index.html локально).

Перетащите файл выгрузки из РосТендера в область загрузки или нажмите на неё для выбора файла.

Дождитесь сообщения об успешной обработке с указанием количества тендеров.

Нажмите кнопку «Скачать CSV».

Импортируйте полученный deal.csv в вашу CRM.

Формат входного файла
Ожидается Excel‑файл (.xlsx или .xls) с первым листом, имеющим структуру:

Строка 1–2 – служебная информация (объединённые ячейки, игнорируются).

Строка 3 – заголовки столбцов. Должны присутствовать следующие заголовки (точно как в выгрузке РосТендера):

Предмет тендера

Заказчик

Контактное лицо организатора (необязательно)

Начальная цена, руб

Ссылка на источник

Дата окончания (МСК)

Строка 4 и далее – данные тендеров.

Формат выходного файла (deal.csv)
CSV с разделителем ; и кодировкой UTF-8 с BOM (для корректного отображения кириллицы в Excel).
Заголовок:

text

Требования к браузеру
Современные версии Chrome, Firefox, Safari, Edge.
Internet Explorer не поддерживается.

Возможные проблемы и ограничения
Если в файле отсутствуют обязательные столбцы, приложение покажет ошибку.

Если столбец с датой содержит текст, не являющийся датой Excel, он останется как есть (без переформатирования).

Гиперссылки извлекаются только из поля «Ссылка на источник», другие ссылки игнорируются.

В сумме удаляются все пробелы, запятая заменяется на точку, затем число форматируется с двумя знаками после запятой. Если преобразование невозможно, поле остаётся пустым.
